JavaNotes 智能学习助手

一个基于 Flutter、Spring Boot 和大模型能力构建的 Java 学习生态系统，覆盖课程学习、AI 助教、在线代码沙盒、算法可视化、错题复盘、社区互动与本地知识库。

Flutter 3.x Spring Boot 3.x Java 17 PostgreSQL Redis AI Assistant

查看架构设计、技术栈、核心功能实现、数据模型与部署运行说明。

功能展示

查看 A/B 学习路径、AI 助教、每日一练、代码沙盒、算法可视化和笔记系统介绍。

项目组成

  • todo_mobile：Flutter 移动端，负责学习路径、AI 对话、笔记、练习和个人中心。
  • todo_service：Spring Boot 后端，负责课程 API、认证、评论、代码运行、AI 调用和数据持久化。

核心能力

  • A/B 双路径学习：零基础闯关路径与进阶项目/算法路径并行。
  • AI 智能助教：支持上下文问答、代码讲解、纠错建议与每日练习生成。
  • 在线代码沙盒：在移动端编写并运行 Java 代码，提升学练闭环效率。
  • 算法可视化：通过 WebView 加载动态演示资源，辅助理解复杂流程。
  • 本地知识库：离线优先的笔记系统，支持学习内容沉淀。

开源版本已移除公网服务器地址、数据库密码、AI Key、JWT Secret 等敏感配置。运行时请通过环境变量或 --dart-define 注入本地配置。
