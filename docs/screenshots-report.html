JavaNotes 功能展示

本页整理 JavaNotes 的主要功能模块，用于开源首页展示。原截图/GIF 中的公网资源地址已在开源版本中移除。

如需展示真实 GIF 或截图，建议将脱敏后的图片资源放入 docs/assets/ 后再引用。

学习路径

A/B 双学习路径

A 路径面向零基础用户，提供循序渐进的闯关式学习；B 路径面向进阶用户，提供项目实战、算法训练和技术文章。

AI 核心能力

AI 助教

支持围绕课程上下文进行连续对话、代码解释、错误分析和学习建议。

每日一练

用户可选择难度、知识点和题型，由 AI 生成个性化练习，并在完成后进入错题回顾与统计分析。

代码沙盒

移动端 Java 运行

后端通过 Java Compiler API 实现代码编译与执行，让用户在手机端完成“学习、编写、运行、反馈”的闭环。

算法可视化

动态算法演示

通过 WebView 承载算法动画资源，帮助用户理解链表、数组、树、动态规划等常见算法过程。

个人知识库与社区

本地笔记

支持 Markdown 笔记、分类管理和本地持久化，方便沉淀学习材料。

评论、收藏与徽章

通过评论互动、内容收藏、学习徽章和错题本构建更完整的学习激励体系。
