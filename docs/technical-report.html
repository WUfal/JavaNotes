JavaNotes 智能学习助手技术报告

JavaNotes 是一款面向 Java 自学场景的移动端应用，集成 AI 智能助教、在线代码沙盒、社区互动、本地知识库和算法可视化能力。

此开源报告为脱敏版本，已移除公网 IP、APK 下载地址、数据库密码、AI Key 和 JWT Secret。

项目介绍

系统采用 A/B 双路径设计：A 路径面向零基础用户，提供闯关式学习体验；B 路径面向进阶用户，提供企业级项目实战、算法可视化演示与深度技术文章。

技术栈

前端

技术说明
Flutter跨平台 UI 框架，实现 Android/iOS 双端一致体验。
Provider全局状态管理，包括用户会话、主题、AI 上下文等。
Sqflite本地优先策略，用于存储离线笔记与刷题历史。
WebView Flutter集成 HTML/JS 静态资源，实现算法可视化动画。

后端

技术说明
Spring Boot 3.x核心业务框架，提供 RESTful API。
PostgreSQL存储课程内容、用户数据与社区评论。
Flyway数据库版本迁移管理。
Spring Security集成 JWT 实现无状态认证。
Java Compiler API服务端动态编译运行用户代码。

系统架构

todo_mobile/
  lib/
    providers/   状态管理
    services/    API 服务
    pages/       UI 页面
    models/      数据模型

todo_service/
  src/main/java/
    controller/  REST 接口
    service/     业务逻辑
    entity/      JPA 实体
    config/      安全与缓存配置
  src/main/resources/
    db/migration/ Flyway SQL 脚本
    static/viz/   算法可视化静态资源

核心功能实现

AI 智能助教与每日一练

后端将耗时的大模型生成流程设计为异步任务，先返回任务 ID，再由后台线程完成题目生成、状态更新和结果持久化，避免移动端长时间阻塞。

个人知识库

笔记系统采用离线优先架构，用户笔记存储在本地 SQLite 数据库中，无需网络也可查看和编辑。

算法可视化

算法动画以静态资源形式放在后端，Flutter 端通过 WebView 加载，并根据运行环境自动拼接服务地址。

部署与运行

后端通过环境变量配置敏感信息；移动端通过 --dart-define=API_BASE_URL=... 注入后端地址。

mvn spring-boot:run
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:8080
